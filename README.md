===================================
PhantomChat - Secure P2P Messenger
=======================

PhantomChat is a ptp chat and file-sharing application 
with ete encryption.

---------------------------
Installation
---------------------------
1. Download the PhantomChat release.
3. Run PhantomChat.exe to start.

---------------------------
Usage
---------------------------
Available Commands:

- add               → Add a peer manually
- list              → Show all peers
- creategroup       → Create a new group
- addtogroup        → Add a member to your group
- removefromgroup   → Remove a member from your group
- leavegroup        → Leave a group
- listgroups        → Show all groups
- send              → Send a message to a peer
- sendgroup         → Send a message to a group
- sendfile          → Send a file to a peer or group
- quit              → Exit PhantomChat

---------------------------
Configuration Files
---------------------------
- config/mykeys.json      → Your private/public key pair
- config/peers.json       → Known peers
- config/queue.json       → Offline messages
- config/groups.json      → Group info
- received_files/         → Stores received files

---------------------------
Security Notes
---------------------------
- All communication is ete encrypted.
- Keep `mykeys.json` private.
- LAN discovery broadcasts your public key only.
